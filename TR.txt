Open Transaction 

사용 테이블

BSID : 고객에 대한 미결(채권)을 관리하는 테이블(반제가 되면 삭제되고, BSAD로 이관)
BSAD : 고객에 대한 반제를 관리하는 테이블

필요 칼럼 

AUGDT : 반제일
KUNNER = Customer Account Code
BSCHL = Transaction Type Code
BUZEI = Transaction Line
AMOUNT = Functional Amount
DOCUMENT_AMOUNT = Document Amount
WAERS = Document Currency Code
BUKRS = Business Unit
HKONT = Subledger Name
BLDAT = Document Date
CPUDT = Entry Date
ZFBDT = Transaction Due Date
BUDAT = Effective Date
FISCAL_PERIOD = Fiscal Period
BELNR = JE Number
AS_OF_DATE = as of date
TRANSACTION_ID = Transaction Reference
SGTXT = Description


Customer Master

사용 테이블

OpenTransaction : 위에서 만든 것

T001 : Company Code
T052 : Terms of Payment
T052U : Payment Description
KNKK : Customer master credit management
KNA1 : General Data in Customer Master
